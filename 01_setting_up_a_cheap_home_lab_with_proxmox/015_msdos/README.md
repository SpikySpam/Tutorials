## Installation

- **Upload** the ISO directly to the Proxmox machine by choosing **Download from URL**
- **Download MS-DOS** from [here](https://www.allbootdisks.com/disk_images/DOS6.22_bootdisk.iso)
- **Create** a new **VM**:
  - Select ISO-file
  - Other Windows as Guest OS
  - Qemu Agent
  - 250MB Disk Size
  - 1024 RAM
- **Start** the VM
